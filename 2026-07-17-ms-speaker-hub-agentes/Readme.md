# 🤖 Agentes Autónomos con IA

## 17 de Julio del 2026

Bienvenido al repositorio de la charla **"Agentes Autónomos con IA"**.

En esta sesión exploraremos qué son los agentes de inteligencia artificial, la diferencia entre agentes conversacionales y agentes autónomos, y cómo podemos construir soluciones avanzadas utilizando modelos de IA y herramientas externas.

Durante la charla veremos:

- Conceptos fundamentales de agentes de IA.
- Diferencias entre agentes conversacionales y autónomos.
- Ejemplo práctico de un Browser Agent.
- Uso de Microsoft Azure AI Foundry.
- Tool Use / Function Calling con modelos de IA.
